# Felizzo'25 Event Dashboard

Web-based dashboard to track and manage Felizzo'25 annual event with weekly pre-events.

## Features
- Real-time event tracking
- Progress dashboard with statistics
- Update events (dates, status, participants, winners)
- Mobile-friendly interface

## Local Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

Access at: http://localhost:5000
