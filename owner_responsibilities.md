# Event Owner Responsibilities

## Owner Checklist for Each Event

### Pre-Event (1 week before)
- [ ] Announce event date and registration deadline
- [ ] Create registration form/process
- [ ] Define rules and format
- [ ] Arrange venue/equipment
- [ ] Communicate with participants

### During Event
- [ ] Set up venue
- [ ] Manage registrations
- [ ] Conduct rounds (prelims, semis, finals if applicable)
- [ ] Track scores/results
- [ ] Take photos/videos for documentation

### Post-Event
- [ ] Announce winners
- [ ] Submit results to organizing committee
- [ ] Share photos/highlights
- [ ] Prepare for final event day (if needed)
